# Grocery Store Billing System (Python)

## Problem Statement
A grocery store wants to calculate the total cost of 3 items purchased by a customer.
If the total exceeds $50, a 10% discount should be applied.
The program should display the original total, discount, and final payable amount.

## Approach
- Take input for 3 item prices using float.
- Calculate total cost.
- Use if condition to check if total > 50.
- Apply 10% discount if applicable.
- Display original total, discount, and final payable amount.

## Sample Output

Enter price of item 1: 20
Enter price of item 2: 25
Enter price of item 3: 15

----- BILL SUMMARY -----
Original Total: $60.00
Discount Applied: $6.00
Final Payable Amount: $54.00
